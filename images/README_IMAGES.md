## 🎨 图片命名规则

### 基本格式
```
立绘_角色名_类型.png
```

### 类型说明
- `_1` = 精一立绘
- `_2` = 精二立绘
- `_skin1` = 第一个皮肤
- `_skin2` = 第二个皮肤
- 以此类推...

### 示例
- ✅ 正确：`立绘_哈蒂娅_1.png`
- ✅ 正确：`立绘_卡涅利安_skin2.png`
- ❌ 错误：`hatia_elite1.png`
- ❌ 错误：`立绘-哈蒂娅-1.png`

## 📐 图片建议规格

- **格式**：PNG（支持透明背景）或 JPG
- **尺寸**：建议 600x800 像素或更高
- **比例**：竖版为佳（3:4 或 2:3）
- **大小**：单张图片建议不超过 500KB

## 🔧 如果没有图片

当前版本使用了占位符显示，即使没有图片也可以正常使用。

如果你想添加真实图片后显示，需要修改 `script.js` 中的 `createOperatorCard` 函数。

详细步骤请查看 `DEPLOYMENT.md` 中的"添加立绘图片"部分。

## 💡 获取立绘的途径

1. **PRTS Wiki**：https://prts.wiki/
2. **明日方舟官方**：游戏内截图
3. **官方社交媒体**：微博、B站等

⚠️ **注意**：请确保你有权使用这些图片，仅供个人学习和娱乐使用。

## 🎯 快速测试

即使没有图片，你也可以：
1. 直接打开 `index.html`
2. 选择干员和立绘
3. 提交投票
4. 查看统计

图片只是增强视觉效果，不影响核心功能！
